Kind-pyplot: Displaying Python plots with the Clojure Kindly convention

Kind-pyplot is a small Clojure library for visualizing Python plots. It uses the Kindly convention, and will thus work in Kindly-compatible tools such as Clay.

The implementation is inspired by the Parens for Pyplot tutorial by Carin Meier from Jan 2020.

Setup

(ns index
  (:require [scicloj.kind-pyplot.v1.api :as pyplot]))

API

The with-pyplot macro takes Clojure forms that create a plot. It evaluates them, and returns a showable (SVG) plot. For example:

(pyplot/with-pyplot
  (matplotlib.pyplot/plot
   [1 2 3 4 5 6]
   [1 4 9 16 25 36]))

The pyplot function takes a Clojure function that creates a plot. It calls the function, and returns a showable (SVG) plot. For example:

(pyplot/pyplot
 #(matplotlib.pyplot/plot
   [1 2 3 4 5 6]
   [1 4 9 16 25 36]))

More examples

From the Parens for Pyplot tutorial

(require '[libpython-clj2.require :refer [require-python]]
         '[clojure.math :as math])
(require-python '[numpy :as np])
:ok
(def sine-data
  (let [x (range 0 (* 3 np/pi) 0.1)]
    (-> {:x (vec x)
         :y (mapv math/sin x)})))
(pyplot/with-pyplot
  (matplotlib.pyplot/plot
   (:x sine-data)
   (:y sine-data)))
(pyplot/pyplot
 #(matplotlib.pyplot/plot
   (:x sine-data)
   (:y sine-data)))
